  2. class
  3. accesskey:
  4. contenteditable 设置元素可修改,浏览器兼容问题,各浏览器展示不一样:

    这个可以修改

  5. dir 设置文字方向 ,rtl,ltr
  6. hidden
  7. lang 局部设置语言
  8. title tips
  9. tabindex 设置 tab 顺序
  10. style 属性

选择器

  1. + 相邻元素选择器
  2. ~ 兄弟元素选择器
  3. > 直接子元素选择器

伪元素选择器

  1. 块级首行选择器 ::first-line
  2. 块级首个字母 ::first-latter
  3. 文本前插入,后插入 ::before,::after 配合 content 最常用于清除浮动

结构性伪类选择器

  1. 根元素选择器 :root 返回 html 元素,不用
  2. 子元素选择器
    1. :first-child 第一个子元素
    2. :last-child 最后一个子元素
    3. :only-child 只有一个子元素的子元素
    4. :only-of-type 子元素只有一种类型的子元素,只会选中第一个
  3. :nth-child(2) 从 1 开始
  4. :nth-last-child(2) 倒数第 2 个
  5. 常用于选中表单元素
    1. :enabled 选中启用的
    2. :disabled 选中禁用的
    3. :checked 勾选的
    4. :default 用得少
    5. :valid 合法的样式
    6. :invalid 不合法的
    7. :required 有必填属性
    8. :optional 非必填属性
  6. 动态伪类
    1. :link 没有被访问
    2. :visited 访问过了
    3. :hover 悬浮状态
    4. :active 激活状态,点下去不松
    5. :focus 聚集时样式
  7. 其它伪类
    1. :not() 里面再写其它选择器
    2. :empty 没有任何内容的元素
    3. :lang(en) 选中英文的
    4. :target 定位到之后选定(锚点定位之后再选中)
